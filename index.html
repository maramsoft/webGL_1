Welcome to MaramSoft 3D maker Demo

The whole idea about the project is how to use WebGL through Javascript libraries, the one I am using is Three.js, which is a very mature library, we use it on the text and convert it to 3D scene without using any traditional 3D programs or plugin, which are difficult to use and maintain, and take a lot of computer memory and resources.

you can start creating a 3D text with the following steps:

  • Start typing to enter the text you want. You can delete existing text using backspace.
  • Change the opacity and depth if you wish form the toolbar in the right side of the window.
  • Change the color of the text and/or background.
  • Resize the text to set the size you want.
  • Use the correct buttons to apply animation on the text
Note: By dragging the text itself you can change the view angle of the text. You can also add a shadow and drag it to the required position, and drag the text to get the angle you want.
